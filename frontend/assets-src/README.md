# assets-src

Source artwork that is **not** shipped to visitors. Nothing in this folder is
served — it lives outside `public/` on purpose, so it stays in git without
adding weight to every deploy.

## Pet Games — Messi photos

| File | What it is |
|---|---|
| `messi-sit-original.png` | Messi sitting. Background already removed. 384×512. |
| `messi-paw-original.png` | Messi with a front paw raised. Background already removed. 433×577. |

These are the originals behind the two frames the Treat Hunt game animates
between (`frontend/public/spike/messi-sit.png` and `messi-paw.png`).

**Why the shipped copies differ:** the two originals have different canvas
sizes and Messi sits in a different spot in each, so swapping them directly
made him jump. The shipped pair was generated by:

1. finding each photo's true content bounds (non-transparent pixels),
2. anchoring on his **feet** — not the overall bounding box, because the raised
   paw juts out to the left and would skew the centring,
3. scaling both so his body is the same height, and drawing each onto a shared
   460×440 canvas with the feet centred.

Keep these originals if you ever want to redo that alignment at a different
size or framing, or add a third pose. Regenerating is a canvas operation —
see the git history for `PetGames.tsx` around 2026-07-20.
